Greatest Innovation

First working programmable computer, the Z3 (1941) made by Konrad Zuse

  • The Z3 was the first programmable computer and fully automatic digital computer. The program code was stored on a punched film. The initial values had to be entered manually. This was a major innovation because it was the first step to the computer programming that we know today.
  • Z3 wikipedia link

First high-level programming language, FORTRAN (1954) made by John Backus

  • FORTRAN was the first high level programming language. FORTRAN was the ideal language for scientific computing and numerical computation. The name FORTRAN stands for Formula Translating System. This was innovative because this the first language that led to the creation of the other popular high level languages like Java, Python, and C++.
  • Link to FORTRAN article

Development of GitHub (2008) made by Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon using Ruby on Rails

  • I think github is one of the most innovative inventions in computing because it was easily allowed programmers to develop and share the code with their team. I started using github in my Object Oriented Programming class with Ray Klump (amazing professor by the way). During this class I realized just how useful it is for saving and uploading your code.
